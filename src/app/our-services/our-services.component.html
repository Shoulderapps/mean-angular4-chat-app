Our Services

Administrator mode

Chatbot will divert user to CS human agent to serve these call in admin mode

Agent need to choose any of the call with “New” status to serve, agent need to input their user name for recording purpose

Agent requires to hit the “exit” button to leave the call in finishing any conversation with user

Agent can handle only one call at any time


Operator mode

Agent can proactively contact any of the customers in operator mode

Agent need to input their user name for recording purpose

Agent requires to hit the “exit” button to leave the call in finishing any conversation with user

Agent can handle only one call at any time


Chat history

Agent can enter the phone # (WhatsApp), or user nickname (WeChat, Line) to identify user you want to check the chat history

Agent need to hit “Exit” button in finishing the action to review chat history


User List

Optional – these feature is available to customer who is in evaluation phase or pay for the value-added service after the evaluation period expire

User list – verified customers, White list – user whom contact chatbot for enquires but not verify yet

CS can add new user (white list) into their own customer contact list (user list) and control the value-added services offer to them

For the sick of evaluation, currently all value-added services are enabled for white list. However, some features can only be available to one on user list only (e.g. set default language in translation).

CS need to specify the type of service enable for every users on user list, e.g. namecard, translation, otherwise, the services will be disabled automatically

CS agent can export the user list in .csv format for broadcasting marketing message purpose
