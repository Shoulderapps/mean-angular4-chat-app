Tips in talking to Mr. Chat

You can use either English or Chinese to ask your questions

If Mr. Chat cannot understand your question twice, Mr. Chat will divert you to human agent

Or you can choose to type “human” or “operator” to ask Mr. Chat to divert you to a human agent

In either case, Mr. Chat will inform you about the status of human agent – either queuing for service or unavailable. In case of queuing, you will be place in work queue to wait for human agent to pick up the call, updated queue status will be sent to you or you can type quit to go back to talk to Mr. Chat.

You can talk to human agent via text / emoji or you can send image (jpg, bmp, gif) with compressed file size less than 1M
